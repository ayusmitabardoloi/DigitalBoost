# DigitalBoost 🚀

A modern, highly responsive landing page for a digital marketing agency, built using a clean and interactive **Bento Grid** user interface.

## 📖 Overview
DigitalBoost is a modern landing page designed to showcase digital marketing services, build brand credibility with dynamic statistics, and convert visitors into leads via an interactive contact form. The layout uses an asymmetrical Bento Grid to create visual interest and organize information naturally.

## ✨ Features
* **Bento Grid Layout**: Structured content organization with smooth hover states and modern elevation treatments.
* **Responsive Design**: Optimized for seamless display across mobile, tablet, and desktop devices.
* **Animated Counter**: Numbers increment dynamically to showcase key statistics as you scroll them into view.
* **Accessiblity**: Follows web standards (semantic markup, clear focus rings, and tab key support).
* **Preloader**: Visual loading screen that automatically fades out once the page assets have loaded.

## 🛠️ Technologies Used
* **HTML5**: Structured semantically for optimal search engine indexing.
* **CSS3**: Layout built with modern CSS Grid and Flexbox, configured using custom properties (CSS variables).
* **JavaScript (ES6+)**: Handles custom element transitions, dynamic counters, and form submission.

## 🚀 Getting Started

### Running Locally
To run the project locally, start a simple HTTP server in this directory:

* **Python 3:**
  ```bash
  python -m http.server 8080
  ```
* **Node.js (npx):**
  ```bash
  npx http-server -p 8080
  ```

Open your browser and navigate to `http://localhost:8080` to view the page.
